# わんちゃん成長日記

うちの子の毎日を、もっと愛おしく。

わんちゃんの成長を記録する日記アプリです。

## 機能

- **日記** - 毎日の出来事や写真を記録
- **体重管理** - 成長グラフで体重の推移を確認
- **症状ログ** - 体調の変化を記録
- **アルバム** - 写真を一覧表示
- **AI健康相談** (プレミアム) - AIによる健康アドバイス
- **犬友つながり** (プレミアム) - 犬友とのソーシャル機能
- **思い出アルバム** (プレミアム) - 思い出をまとめてPDF出力

## 技術構成

- React (シングルファイルビルド)
- PWA対応 (オフライン対応、ホーム画面追加)
- LocalStorage によるデータ保存
- GitHub Pages でホスティング
