# Consulting Playbook

## 案件フェーズ

### Phase 1: Discovery（課題発見）
1. クライアントの事業構造を把握（売上構成、コスト構造、利益率）
2. 市場環境の分析（TAM/SAM/SOM、競合、トレンド）
3. 現状のKPI体系と課題仮説の整理
4. **佐藤裕介チェック**: PL上のどこが最大のレバレッジポイントか特定

### Phase 2: Analysis（分析）
1. データに基づく課題の構造化
2. 根本原因の特定（表面的な症状ではなく）
3. ベンチマーク比較（業界標準、競合、過去実績）
4. **小野寺チェック**: その指標は本当に事業成長に直結しているか

### Phase 3: Strategy（戦略策定）
1. 戦略オプションの策定（最低3案）
2. 各オプションのPLシミュレーション
3. トレードオフの明示
4. 推奨案の選定と根拠

### Phase 4: Execution Plan（実行計画）
1. アクションアイテムの洗い出し
2. 担当×期限×KPIの設定
3. リスクと対策の整理
4. 効果測定の方法

## 品質チェックリスト
- [ ] 結論が一文で言えるか
- [ ] 全ての主張に数値的根拠があるか
- [ ] PLインパクトが試算されているか
- [ ] トレードオフ・リスクが明示されているか
- [ ] 具体的なアクション（誰が/何を/いつまでに）があるか
- [ ] 抽象的な表現が残っていないか
