# Creative Playbook

## 制作プロセス

### Phase 1: ブリーフ
- 目的: 何のためのクリエイティブか（KPIへの接続）
- ターゲット: 誰に届けるか（ペルソナ×文脈）
- メッセージ: 何を伝えるか（1つに絞る）
- トーン: どう伝えるか（ブランドガイドラインに準拠）
- 制約: 予算・納期・技術的制約

### Phase 2: コンセプト開発
- 最低3案のコンセプトを提示
- 各案のメリット/デメリット/コスト比較
- **佐藤裕介チェック**: 既存アセットの活用は検討したか

### Phase 3: 制作
- ブランドガイドラインに準拠
- モバイルファーストで制作
- ダークモード対応を忘れない
- アクセシビリティ（WCAG 2.1 AA）準拠

### Phase 4: レビュー・承認
- brand-guardianによるガイドライン適合チェック
- creative-directorによる品質レビュー
- kpi-analyticsによる効果予測

## 品質チェックリスト
- [ ] ブランドカラー・フォントが正確か
- [ ] 1メッセージに絞れているか
- [ ] ターゲットの文脈に合っているか（小野寺: 文脈設計）
- [ ] CTAが明確で自然か（佐藤裕介: 売りつけない）
- [ ] モバイル/デスクトップ両方で表示確認したか
- [ ] ダークモードで問題ないか
- [ ] 画像の代替テキストがあるか

## リパーパス設計（佐藤裕介: アセット優先）
1つの制作物から最大限の展開を設計:
```
ブログ記事 → SNS投稿(3-5件) → メルマガ → ウェビナー資料 → 営業資料
```
