## A/Bテスト設計（3本）

### テスト1: 足あと vs ランキング
- **仮説**: 温かい接点が7日リテンションを+5pt改善
- **サンプル**: 1,966人（983/群）
- **期間**: 28日間
- **成功基準**: 7日リテンション +5pt、p < 0.05

### テスト2: 「犬友X人」vs フォロー数/フォロワー数
- **仮説**: 対称表示が犬友申請率+30%改善
- **サンプル**: 7,552イベント
- **期間**: 14日間
- **成功基準**: 申請率 +30%、p < 0.05

### テスト3: 横カルーセル vs 縦リスト
- **仮説**: カルーセルがプロフィール閲覧数+50%
- **サンプル**: 1,000人
- **期間**: 7日間
- **成功基準**: 閲覧数 +50%、p < 0.05

### 実施スケジュール
```
Week 1-4: テスト1（最重要）
Week 2-3: テスト3（並行実施可）
Week 5-6: テスト2（テスト1結果依存）
Week 7:   統合分析
```
