# わんちゃん成長日記 ブランドガイドライン（包括版）

## 1. ブランドパーソナリティ

### ミッション
飼い主とわんちゃんの毎日を、もっと楽しく、もっと安心に記録する。

### コンセプト
「うちの子自慢帳をみんなで回す」— SNS的競争ではなく、mixi的な温かい接点

### パーソナリティ4原則
- **あたたかい**: 飼い主の愛情に寄り添う。数値で競わせない
- **やさしい**: 難しい機能も簡単に使える。専門用語を使わない
- **信頼できる**: 大切な記録を安全に守る。AI相談は必ず免責表示
- **楽しい**: 日記をつけること自体が楽しい体験

### NGワード・NG表現
| NG | OK代替 |
|---|---|
| ペット、犬 | うちの子、わんちゃん |
| フォロー/フォロワー | 犬友 |
| エントリー、投稿 | 日記 |
| ランキング、競争系表現 | 足あと、ひとこと |
| エラーが発生しました | もう一度試してみてね |
| データベース、API、キャッシュ | （使わない） |
| 「犬を飼っている」 | 「わんちゃんと暮らしている」 |

## 2. ビジュアルガイドライン

### カラーパレット
| 用途 | 変数 | ライト | ダーク |
|---|---|---|---|
| 背景 | bg | #FFF8F5 | #121212 |
| カード | c | #FFFFFF | #1E1E1E |
| サブ背景 | s | #FFF0EB | #2A2A2A |
| アクセント | ac | #FF7B9C | #FF7B9C |
| 薄アクセント | a2 | #FFF0F3 | #2A1A22 |
| テキスト | tx | #3D2C2C | #F0F0F0 |
| サブテキスト | dm | #BFA6A0 | #BBB |
| ボーダー | bd | #F0D9D1 | #3A3A3A |
| グラデーション | g1/g2 | #F5A6B8/#FF7B9C | 同左 |

**使用ルール**:
- アクセント色(ac)はCTA、アクティブタブ、重要アクションのみに使用
- グラデーション(g1/g2)は投稿ボタンのみに限定使用
- 背景にグラデーションは使わない（単色で落ち着いた印象を維持）

### タイポグラフィ
| 用途 | サイズ | ウェイト | 行間 |
|---|---|---|---|
| 画面タイトル | 16px | 800 | 1.4 |
| セクション見出し | 14px | 700 | 1.4 |
| 投稿タイトル | 15px | 700 | 1.4 |
| 本文 | 13px | 400 | 1.7 |
| 補助テキスト | 11px | 400 | 1.4 |
| ボタン内 | 12px | 700 | 1.0 |

- フォント: Zen Maru Gothic（丸ゴシック）→ Hiragino Sans → sans-serif

### 角丸・余白・影
| 要素 | 角丸 | 余白 |
|---|---|---|
| カード | 16px | padding: 14px |
| ボタン | 20px | padding: 6px 18px |
| アバター | 50% | - |
| 入力フィールド | 12px | padding: 10px 14px |
| タグ/バッジ | 12px | padding: 4px 10px |

- 画面パディング: 16px
- カード間: marginBottom 12px
- セクション間: marginBottom 20px
- 影: boxShadow "0 2px 8px rgba(0,0,0,0.06)" （カードのみ、控えめに）

### アイコン・絵文字
- 絵文字ベース優先（カスタムアイコンより親しみやすさ重視）
- 主要絵文字: 🐾(足あと) 🐶(わんちゃん) 📝(日記) 📷(写真) 💬(コメント) ❤️(いいね) 🩹(健康)

## 3. コピーライティングガイドライン

### 文体ルール
- **です/ます調**: 丁寧だけど堅すぎない
- **語尾バリエーション**: 〜だよ、〜してみてね、〜しよう
- **飼い主目線**: 「あなたのわんちゃん」ではなく「○○ちゃん」（名前で呼ぶ）

### OK/NG例
| 場面 | OK | NG |
|---|---|---|
| 日記投稿促進 | 「今日のうちの子の様子を記録しよう」 | 「日記エントリーを入力してください」 |
| 保存完了 | 「素敵な記録が増えたよ!」 | 「データが保存されました」 |
| エラー時 | 「うまくいかなかったみたい。もう一度試してみてね」 | 「エラーが発生しました (Code: 500)」 |
| 足あと通知 | 「○○さんが遊びに来てくれたよ!」 | 「新しいフットプリントがあります」 |
| 犬友申請 | 「犬友になる」 | 「フォローする」 |
| プレミアム訴求 | 「もっと楽しくなる機能を見てみる?」 | 「プレミアムプランにアップグレード」 |
