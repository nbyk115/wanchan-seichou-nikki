# わんちゃん成長日記 Go-to-Market戦略 / 事業成長戦略

## 【結論】
月額480円のフリーミアムモデルで、初期6ヶ月はSNS口コミ+犬種特化コミュニティを起点にMAU 30,000を獲得し、課金転換率5%で月間MRR 72万円を目指す。12ヶ月で損益分岐点（月間MRR 120万円）到達が目標ライン。

## 1. 市場構造分析

### TAM / SAM / SOM

| 区分 | 定義 | 算出根拠 | 規模 |
|------|------|----------|------|
| **TAM** | 日本の犬飼い主全体 x アプリ利用可能性 | 犬飼育頭数705万頭 x 飼い主世帯約500万世帯 x 年間ARPU 5,760円 | **約288億円/年** |
| **SAM** | スマホ利用+20-40代女性犬飼い主 | 500万世帯のうちスマホ活用層かつ20-40代女性が主導する世帯 約35%（175万人） x 年間ARPU 5,760円 | **約101億円/年** |
| **SOM** | 12ヶ月で獲得可能な現実的シェア | MAU 5万人 x 課金率5% x 年間ARPU 5,760円 | **約1,440万円/年** |

**TAM算出ロジック**: ペット関連市場1.7兆円のうち、ペットテック(アプリ/デジタルサービス)は市場全体の約1.7%と推定。犬特化の記録/SNSアプリ領域に限定。

### 自社の差別化ポイント（3つのWedge）

1. **「犬友ひろば」= 犬種特化SNS**: 犬種別コミュニティは既存競合にない。犬種ごとの悩みは共通性が高く、ネットワーク効果が期待できる
2. **AI健康相談**: 「うちの子が急にご飯を食べない」等の不安に即時回答。獣医に行く前のファーストタッチを押さえる。無料枠5回/月 → 課金動機に直結
3. **PWA x 低コスト運営**: ネイティブアプリではなくPWA。App Store手数料30%を回避し、開発・運用コストを最小化。この構造的コスト優位が価格競争力の源泉

### 価格帯の根拠

**競合比較**:
| サービス | 価格 | 提供内容 |
|----------|------|----------|
| いぬのきもち | 1,078円/月 | 情報メディア+一部記録機能 |
| Findog | 500円/月 | GPS+健康管理 |
| 犬関連サブスク平均 | 500-1,500円/月 | 各種 |
| **わんちゃん成長日記** | **480円/月** | 記録+SNS+AI相談 |

**WTP（支払意思額）分析**:
- ターゲット層（20-40代女性犬飼い主）の月間ペット関連支出: 平均15,000-30,000円
- デジタルサービスへの支出許容: 月間ペット支出の1-3% = 150-900円
- 480円/月は許容レンジの下位に位置 → **価格障壁を最小化し、転換率を最大化する戦略**
- 将来的に機能追加に伴い680円への値上げ余地あり

**年間プラン設計（現行）**:
| プラン | 価格 | 月換算 | 割引率 |
|--------|------|--------|--------|
| 月額 | 480円 | 480円 | - |
| 年額 | 3,980円 | 約332円 | **31%OFF** |

年間プラン割引率31%は妥当。目標: 課金ユーザーの30%を年間プランに誘導 → LTV向上+チャーン低減。

## 5. リスクと対策

### 主要リスク3つ

#### リスク1: ネットワーク効果の未起動（犬友ひろばが過疎化）
- **発生確率**: 高（コミュニティ系サービスの最大の死因）
- **影響**: MAU成長の停滞、課金転換率未達
- **対策**:
  - 犬種別に「最低30人/犬種」が集まるまで、運営側がシードコンテンツを投稿（1日10件 x 30日）
  - Phase 1でインフルエンサー10名を「公式犬友アンバサダー」として起用し、初期コンテンツ密度を確保
  - 犬種別の閾値（アクティブ投稿者30人）未達の犬種は一時的に統合表示し、過疎感を回避
- **判定指標**: M2末時点で犬友ひろば投稿数 500件/月未達 → 施策見直し

#### リスク2: AI健康相談のコスト肥大・品質リスク
- **発生確率**: 中（APIコストは利用量に比例、誤情報リスクは常時）
- **影響**: 利益率低下、ユーザー健康被害によるレピュテーションリスク
- **対策**:
  - 全回答に「獣医への相談を推奨」の免責表示を必須化（現在未実装なら即対応）
  - APIコスト上限を月間予算の15%以内に設定。超過時はレスポンスのモデルをダウングレード
  - 回答品質を獣医監修で四半期ごとにサンプルレビュー（100件抽出）
- **判定指標**: API費用がMRRの20%を超過 → 無料枠を月3回に削減

#### リスク3: PWA（非ネイティブアプリ）の限界によるユーザー獲得困難
- **発生確率**: 中-高（App Store/Google Play Storeに掲載されないことの機会損失）
- **影響**: オーガニック獲得チャネルの欠如、「アプリストアで検索して見つける」ユーザーを取りこぼす
- **対策**:
  - M6時点でMAU 12,000未達の場合、React Nativeでネイティブアプリ化を検討（既存PWAコードの80%は再利用可能）
  - PWAのまま行く場合、Google検索経由のWebアプリ流入をSEOで最大化（「犬 成長記録 アプリ」で1ページ目表示を狙う）
  - TWA（Trusted Web Activity）でGoogle Play Storeに掲載する中間策も選択肢
- **判定指標**: M4末でオーガニック流入が全体の20%未満 → TWAまたはネイティブ化を決断

### 撤退判断基準

以下の**いずれか2つ以上**に該当した場合、ピボットまたは撤退を検討する:

| 基準 | 閾値 | 判定時期 |
|------|------|----------|
| MAU成長率 | 3ヶ月連続で月次成長率10%未満 | M4以降 |
| 課金転換率 | M6時点で2%未満 | M6 |
| DAU/MAU比率 | 15%未満が2ヶ月連続 | M3以降 |
| 月間バーンレート | 累計投資額300万円超過でMRR 10万円未満 | M6 |
| NPS | -10以下 | M4以降 |

**撤退時のアセット転用先**:
- 犬種別体重データ/症状データ → ペット保険会社へのデータ販売
- AI健康相談のプロンプト設計 → 獣医クリニック向けB2B SaaSへのピボット
- 犬友ひろばのUGC → ペットメディアへのコンテンツライセンス

## 【推奨案とトレードオフの明示】

**推奨**: 上記Phase 1-3のロードマップをPWAのまま実行し、M6時点で次の判断を行う。

**トレードオフ**:

| メリット | コスト/リスク |
|----------|--------------|
| PWAで初期投資を最小化（アプリストア手数料30%回避） | App Store非掲載によるオーガニック獲得の機会損失 |
| 月額480円の低価格で転換率最大化 | ARPU低のため大量ユーザーが必要（薄利多売構造） |
| AI健康相談が最大の課金ドライバー | APIコスト変動リスク+医療情報の法的リスク |
| 犬友ひろばでネットワーク効果を狙う | コールドスタート問題が最大のハードル |
